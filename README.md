# AK Digital Hub

Welcome to **AK Digital Hub**! 🚀

An interactive application designed to provide top-notch digital solutions.

## 🌟 Features
- Fast, secure, and modern UI built with React & Vite
- Integrated with Firebase for authentication and database
- Cloudinary integration for smooth media management
- Dynamic order tracking and dashboard

## 🛠️ Tech Stack
- Frontend: React, Vite, TailwindCSS
- Backend: Firebase Realtime Database
- Cloud: Cloudinary (Image handling)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm installed.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Pavankumarswamy/ak-digital.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Set up your `.env` file based on `.env.example`
   ```sh
   cp .env.example .env
   ```
   *(Make sure to replace the dummy credentials with your actual service keys.)*

4. Run the development server
   ```sh
   npm run dev
   ```

## 🔐 Environment Variables

This project uses `.env` variables to store API keys. See `.env.example` for the required keys.
**Note:** Never commit your `.env` file to version control. The `.gitignore` is already set up to exclude it!
